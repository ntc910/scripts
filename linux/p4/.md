P4 setup in linux
===============================================================
This guide tested on Ubuntu 22.04
On Arch, just use AUR
### Add perforce repo to apt

```sh
wget -qO - https://package.perforce.com/perforce.pubkey | gpg --dearmor | sudo tee /usr/share/keyrings/perforce.gpg
sudo vim /etc/apt/sources.list.d/perforce.list
```
Add the Perforce repository to your sources list:
```sh
deb [signed-by=/usr/share/keyrings/perforce.gpg] https://package.perforce.com/apt/ubuntu jammy release
```
Then run
```sh
sudo apt update
sudo apt install p4-cli
```

### Setup p4-cli
```sh
p4 set P4USER=<knox-id>
p4 set P4PORT=107.113.53.156:1716
```
Now you can log in with
```sh
p4 login
```

P4V install in wsl (run faster than Windows)
===============================================================
Install
```sh
sudo apt install p4v
```
### Install following components before running p4v
```sh
sudo apt install libqt5gui5 libgl1-mesa-glx libglx-mesa0 libgl1-mesa-dri libglu1-mesa libnss3 libxkbfile1 libxcb-cursor-dev
``` 

### Run p4v
```sh
export DISPLAY=:0
p4v
```
